## 📞 Troubleshooting Tips

**If interns can't clone**
- Check repository visibility (should be public)
- Verify repository URL is correct
- Check they're using HTTPS (not SSH)

**If npm install fails**
- Have them verify Node version: `node --version` (should be v16+)
- Clear npm cache: `npm cache clean --force`
- Delete node_modules and try again

**If npm start fails**
- Check port 3000 isn't in use
- Try: `npm start -- --port 3001`
- Check Vite is properly installed

**If git workflow issues**
- Have them run `git status` to see current state
- Ensure they created feature branch BEFORE editing files
- Guide them through commit process step by step
